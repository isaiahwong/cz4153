# CZ4153 Blockchain

## Running on local node
### Run local eth node
Open a separate shell 
```
$ npm run node
```
### Deploy contracts
Open another shell
```
$ npm run deploy
```
`address.local.json` will be generated in the root folder and 
frontend root `./frontend`

## Starting Frontend
```
$ cd frontend 
$ npm i && npm run start
```

## Running Tests
```
$ npm run test
```
